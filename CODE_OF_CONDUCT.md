# Code of Conduct

ztrack uses the Contributor Covenant Code of Conduct as its community standard.

- Canonical text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
- Project scope: issues, pull requests, discussions, project chat, and any other ztrack community space.
- Reporting contact: use GitHub Issues for non-sensitive conduct concerns. For private safety/security issues, use GitHub Security Advisories.
