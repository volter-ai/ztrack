# Security Policy

ztrack is local-first and sends no telemetry — it runs on your machine and
talks only to the services you point it at (git, your PR host, your tracker).

## Reporting a vulnerability

Please report security issues privately through a GitHub security advisory.

Security email: TODO before launch if a monitored inbox is configured.

We aim to acknowledge within 72 hours once the reporting channel is live.

Do not open public issues for security reports.
