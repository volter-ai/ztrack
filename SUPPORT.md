# Support

## Questions

Open a GitHub issue with the `question` template. Include your install path,
tracker backend, and the command you ran.

## Bugs

Open a bug report with:

- ztrack version or commit SHA
- install path (`npx`, source checkout, GitHub Action)
- smallest reproduction steps
- command output, redacted if needed

## Security

Do not open public issues for vulnerabilities. Use GitHub Security Advisories for
this repository; see [SECURITY.md](SECURITY.md).

## Startup Pilot

Teams that want managed setup and workflow review can apply at
https://ztrack.dev/startup-pilot.html.
